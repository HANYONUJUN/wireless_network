### 프로젝트 아이디어 수립 및 기능제시
> 23.11.07 ~ 23.11.13

1. 산업현장 안전관리 모니터링 카메라
2. 목적
   - 산업현장 같은 곳에서 이런 cctv 장치의 배치가 적게 되어 사고가 발생해도 다음날 발견되는 경우가 많아 그런 부분을 감소시키기 위해서이다.

3. 기능 : 쓰러짐 감지, 특정 행동 감지, 실시간 모니터링 및 로그 누적
   - 쓰러짐 감지 : 근로자가 쓰러짐을 감지하면, 관리자에게 알림을 전송
   - 특정 행동 감지 : 근로자의 특정행동을 감지하여 그에 맞는 알림을 전송한다. (사고 신고 모션인식 등)
   - 로그 누적 : 사소한 감지상태도 로그 누적을 통한 관리
   - 실시간 모니터링 : 감독관이 실시간으로 현장을 사고를 파악
     
4. 구조
   1. 라즈베리파이 + 카메라
   2. 실시간 모니터링 웹페이지 및 앱
   3. API서버(AWS) (알림전송, 모델적용, 기타 기능구현 서버)
   4. vue.js를 통해 cctv내용들의 대한 ui를 구현<br>

> ui 구조
> 1. 실시간 이미지 출력화면
> 2. 실시간 이미지 녹화 시작버튼
> 3. cctv 관리자 정보 등록 (아직 구현 안됨)
> 4. 최근 활동 모션이 감지된 이미지 캡처 후 화면의 띄워주기(아직 구현 안됨)
> 5. 감지상태를 로그로 통환 관리(아직 구현 안됨)
> <h2>모델 적용 후 추가시킬 기능</h2><br>
1. 관리자 정보 입력 후 정보 등록 버튼과 스트리밍 버튼을 누르면 서버로 정보 전송 후 특정 감지에 대한 문자
정보를 보낼 전화번호 지정
2. 특정 행동이 감지되면 감지된 내용에 대한 이미지를 화면에 캡쳐하여 띄워줌
3. 감지에 대한 내역을 로그로 출력 (예: 2023-11-28: 특정 행동 감지)
